About

Freelancer هو سمة تمهيدية مجانية
تم إنشاؤها بواسطة Start Bootstrap. يتضمن التنزيل
ملفات المصدر الكاملة بما في ذلك HTML و CSS و
JavaScript بالإضافة إلى SASS الاختياري
أوراق الأنماط لسهولة التخصيص.

يمكنك إنشاء الصورة الرمزية المخصصة الخاصة بك
بالنسبة لإعلان التسمية الرئيسية، قم بتغيير الرمز الموجود في
المقسمات، وأضف عنوان بريدك الإلكتروني إلى
نموذج الاتصال لجعله يعمل بكامل طاقته!
